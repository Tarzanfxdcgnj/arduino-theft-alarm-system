# Arduino Theft Alarm System – Cybersecurity Application

This project is a theft alarm system built using **Arduino** for the CM1111 – Exploring Computing Devices module. The system demonstrates practical applications of embedded security, monitoring, and alert systems.

## Overview

The objective of this lab was to design a basic physical security system using Arduino. This project links **cybersecurity principles** to embedded systems by showing how sensors, alerts, and user input can help prevent unauthorized access.

## Environment & Components Used

- **Arduino IDE**  
- **Hardware Components:**
  - PIR motion sensor
  - LCD display
  - Buzzer
  - LED
  - Push button
  - Potentiometer
- Tested on a **Tinkercad simulation or real Arduino board**

## Key Tasks Performed

1. Designed an Arduino program to detect unauthorized motion.  
2. Controlled output devices (buzzer, LED) to trigger alarms.  
3. Displayed system status on an LCD to monitor activity.  
4. Integrated multiple sensors for reliable detection.  
5. Tested and debugged the system to ensure security functionality.  
6. Documented the setup with screenshots and diagrams.

- Basic embedded system security
